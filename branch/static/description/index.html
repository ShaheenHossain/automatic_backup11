Multiple Branch(operation) Management Module for single company.

Do you have multiple unit for single company? Do you want them to be work as separate entity inside the company? Here you go, we have make plugin that helps users to make different branch for single company with multi branch concept which works same as multi-company environment.

Also Don't worry about the access rights too, We have added branch user/manager roles inside the module, Branch user can only access records of its specific branch and Branch manager can see records of all Branches.

key features:

  • Able to add multiple branch/unit for single company.
  • We have two access group created in our module which is Branch user and branch manager.
  • Branch user can only access records of its specific branch and it's working as multi branch concept same as Odoo's multi company environment.
  • Branch manager can see all records for whatever branch it is.
  • Branch functionality also added to Warehouse.
  • Branch functionality also added POS module too , Which works based on session and branch is printed on POS receipt.
  • Single Chart of accounts for all branches, but when accounting transaction generated for specific branch.
  • Automatically set selected branch on complete workflow i.e sales to delivery, sales to invoice, purchase to shipment, purchase to invoice, invoice to Journal entry and payment.
  • Branch is also automatically effected on stock journal entry also when product's Inventory valuation set as Automated(Real Time).
  • Branch Wise Financial Statement filter .i.e Balance Sheet, Profit & Loss, General Ledger, Trail Balance etc all reports you can print for specific branch.

How to create multiple branch for company

You can go to Settings > Users > Branch.


Assigning a branch to user

You can assign a branch for which specific user is belongs.


You can change the preference of branch also.

Quotations / Sales Order

While creating sales order user who is currently login, branch of the user is set as branch to the sales order and warehouse shipment address is also set according to selected branch.


Invoices

While invoice created from the sales order selected branch of the sales order will automatically set on invoice.
When creating manual invoice then log in user's branch will be assigned to the invoice.




Branch Name Bank Statements.

While creating Bank Statements you can also show branch details in it. After that while validate it also display in the journal entry of it.


Purchase Order.

While creating purchase order user who is currently login, branch of the user is set as branch to the purchase order.


While creating purchase order warehouse picking address is set according to branch which is selected.


It display warning if you select different branch than warehouse.


Branch on Picking(Shipment/Delivery)

While RFQ(purchase) or quotation get confirmed generated picking(shipment/delivery) will have same branch automatically set based on RFQ/Sales Order.


You can also see the branch name in the journal entries in the product.


If you set Automated(Real time) Inventory valuation for your inventory setup then generated stock accounting entry will also have same branch will be linked which used to print financial statement properly with branch filter.


Warehouse

You can also define branch in the warehouse and all the technical fields are set according to the branch.


In warehouse locations you can also set branch.


It display warning if you select different branch on location than warehouse.


Point Of Sale.

While creating POS orders login user's branch will be pass in the POS session and the orders created by that session you can also see the branch in the POS receipt.

Branch on POS order.

Branch on Payment.

You can see in the session that only allowed branches in POS are display here for selection


Branch Wise Accounting Report.

In wizard of financial report you can select any branch and details to that branch will be displayed.
